# DevSimulate — VS Code Extension

Solve real engineering tickets, get scored by AI, and build a verified skill profile that employers trust.

## Features

- **Login with GitHub** — one-click authentication
- **Clone Codebase** — instantly clone the assigned ticket's repo
- **Submit PR** — open a pull request and trigger AI code review
- **View Score** — see your latest score directly in VS Code
- **Sidebar Panel** — track your ticket assignment and submission status

## How It Works

1. Sign in with your GitHub account
2. Get assigned a real-world engineering ticket
3. Fix the bug or implement the feature in your fork
4. Submit your PR — Claude AI reviews your code
5. Get scored on Diagnosis, Design, Communication, and Execution

## Requirements

- VS Code 1.85 or higher
- A GitHub account
- A DevSimulate account at [devsimulate-mono-web.vercel.app](https://devsimulate-mono-web.vercel.app)

## Extension Settings

- `devsimulate.apiUrl` — DevSimulate API base URL (default: production)
